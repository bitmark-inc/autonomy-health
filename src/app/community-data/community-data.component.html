Submitting data to Berkeley CDE

When you submit data in Autonomy, an anonymized, non-personally-identifiable version will be automatically submitted to the UC Berkeley community data enclave (CDE) for community-level pooling and rsearch. Currently, you can only save Autonomy data to the Berkeley CDE via the Autonomy app interface.

The Autonomy app presently supports the following types of data to be submitted to the Berkeley CDE:

Resource ratings

You can view all Autonomy data you have submitted to the Berkeley CDE, by exercising your right to do so.
